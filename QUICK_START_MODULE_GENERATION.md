  getList(params) {
    return request({
      url: '/orders',
      method: 'get',
      params
    })
  },
  // ... 其他接口
}

// 2. 在组件中引入
import orderApi from '@/api/modules/orderManagement'

// 3. 替换 axios 直接调用
// 修改前：
const response = await axios.get('/api/orders', { params })

// 修改后：
const response = await orderApi.getList(params)

## ❓ 常见问题

### Q1: 生成的代码无法运行？

**A**: 检查以下几点：
1. Vue 和 Element UI 版本是否匹配
2. 路由配置是否正确添加
3. 组件名称是否符合规范
4. 浏览器控制台有无错误信息

---

### Q2: 如何自定义生成的代码？

**A**: 在提示词中明确说明：
```
"在客户管理模块的基础上，添加以下功能：
1. 客户标签管理
2. 客户跟进记录
3. 客户转化率统计"
```

---

### Q3: 一次生成多少个模块合适？

**A**: 建议每次 2-3 个模块，验证通过后再继续。新手建议每次 1 个。

---

### Q4: 生成的代码质量不够高怎么办？

**A**:
1. 检查提示词是否清晰明确
2. 使用质量评估清单找出具体问题
3. 要求 Claude Code 修复特定问题
4. 如果得分 < 70，建议重新生成

---

### Q5: 如何让生成的代码更符合项目风格？

**A**: 在提示词中强调：
```
"严格参考 src/views/modules/CustomerManagement.vue 的代码风格，包括：
- 组件结构布局
- 方法命名方式
- 样式定义方式
- 错误处理方式"
```
